Date/Time Specification

Many functions in the ChartDirector API accept dates/times as parameters. ChartDirector accepts two types of date/time formats:

  • Ruby Time objects

  • ChartDirector time format as seconds elapsed since 1-1-0001 AD 00:00:00. The ChartDirector functions ChartDirector.chartTime and ChartDirector.chartTime2 can be used to construct date/time in this format.
