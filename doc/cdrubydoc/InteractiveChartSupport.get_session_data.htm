InteractiveChartSupport.get_session_data

Usage

get_session_data()

Description

Implements an action for retrieving chart images or delayed image maps from session variables.

The session variable name is taken as the "id" query parameter. This action sends the content to the browser directly, so no separate view file is needed.

Arguments

None

Return Value

None
